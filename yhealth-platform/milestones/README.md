# Milestones

This folder contains individual milestone files for this product.

## File Naming Convention

`M-XXX-descriptive-name.md`

Examples:
- `M-001-established-vision.md`
- `M-002-completed-prd.md`
- `M-003-design-system-review.md`

## Template

Use `FRAMEWORKS/templates/milestone-template.md` when creating new milestones.

## Relationship to PROGRESS.md

PROGRESS.md contains a summary table that references these files:

```markdown
### Completed
| ID | Title | Completed | Link |
|----|-------|-----------|------|
| M-001 | Established vision | 2025-12-20 | [View](./milestones/M-001-established-vision.md) |
```
