# Work Items

This folder contains individual work item files for this product.

## File Naming Convention

`NS-XXX-descriptive-name.md`

Examples:
- `NS-001-complete-e00-documentation.md`
- `NS-002-cloud-evaluation.md`
- `NS-003-cicd-pipeline.md`

## Template

Use `FRAMEWORKS/templates/work-item-template.md` when creating new work items.

## Relationship to NEXT-STEPS.md

NEXT-STEPS.md contains a summary table that references these files:

```markdown
### Currently In Progress
| ID | Title | Priority | Assignee | Link |
|----|-------|----------|----------|------|
| NS-001 | Complete E00 docs | P0 | Hamza | [View](./work-items/NS-001-complete-e00-documentation.md) |
```

## Work Item States

- `queued` - Ready but not started
- `in_progress` - Actively being worked on
- `blocked` - Cannot proceed
- `completed` - All criteria met
